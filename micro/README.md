# Micro - Privacy-First Personal Assistant

Micro is a privacy-first, autonomous agentic mobile assistant built with Flutter and Riverpod. It provides universal capabilities through MCP (Model Context Protocol) integration while maintaining local-first storage and end-to-end encryption.

## Getting Started

### Prerequisites

- Flutter SDK (>=3.2.0)
- Dart SDK (>=3.2.0)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/micro.git
cd micro
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
